Pug SDK — CDN loader snippet

Loads the SDK as the IIFE bundle via the production loader snippet (with src pointed at the local build), queues init() and a track() call before the script loads, and drives everything through window.pug. Runs in dryRun mode with consent denied at start, so nothing is sent. Build first: bun run build && bun run serve, then open http://localhost:3000/examples/cdn/. Watch the devtools console (enable Verbose) for SDK logs.
